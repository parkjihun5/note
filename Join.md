## JOIN
JOIN은 두 개 이상의 테이블을 연결하여 데이터를 조회하는 SQL 구문

### JOIN의 종류
INNER JOIN
    두 테이블에서 조인 조건이 일치하는 레코드만 반환
    INNER 키워드를 제외하여 JOIN만 명시하여도 INNER JOIN으로 동작

### LEFT JOIN
왼쪽 테이블의 모든 레코드와 오른쪽 테이블의 일치하는 레코드 반환

### RIGHT JOIN
오른쪽 테이블의 모든 레코드와 왼쪽 테이블의 일치하는 레코드 반환
    LEFT JOIN의 순서를 바꾸면 되기 때문에 잘 사용하지 않음

### FULL JOIN
왼쪽 테이블의 모든 레코드와 오른쪽 테이블의 모든 레코드를 반환
    양쪽 모두 NULL이 포함 될 수 있음
    모든 데이터를 다 가져오기 때문에 특수한 경우가 아니면 잘 사용하지 않음

### Self JOIN
JOIN의 한 형태는 아니라 같은 테이블을 자기 자신과 조인하는 방식이며, INNER JOIN, LEFT JOIN 등 모두 사용 가능
