## 연산자  
- 산술 연산자: +, -, *, /(몫), %(나머지)
- 비교 연산자: ==, !=, >, <, >=, <=
- 논리 연산자: && (AND), || (OR), ! (NOT)
- 대입 연산자: =, +=, -=, *=, /=, %=

**논리 연산자**
```java
//자료가 int형일 때
int a = 7
int b = 7

a == b //= true
```
```java
//자료가 str형일 때
String s1 = new String("hi")
String s2 = new String("hi")

//string형일 때 ==는 값이 아닌 주소값 비교
s1 == s2 = //false

//string형의 값을 비교하고 싶을 때
s1.equals(s2)
```

## 멤버 변수 vs 지역 변수
**멤버 변수**
   - 클래스 영역에 선언됨
   - 객체와 운명을 같이 함(Heap 메모리에 저장)
   - 기본값으로 자동 초기화됨

**지역 변수**
   - 메서드나 블록 내부에서만 유효함(Stack 메모리에 저장)
   - 블록을 벗어나면 즉시 소멸함
   - **사용 전 반드시 초기화**가 필요함(안 하면 컴파일 에러)

## 형 변환(Casting)
**묵시적 형 변환(Promotion)**
   - 작은 메모리 타입에서 큰 타입으로 자동 변환
   - 예: `long a = 100;` (int가 long으로 자동 승격)
   - 데이터 손실 걱정 없음.

**명시적 형 변환(Demotion)**
   - 큰 메모리 타입에서 작은 타입으로 강제 변환
   - 변수 앞에 `(타입)` 명시 필수
   - 예: `int b = (int)10.5;` (결과는 10, 소수점 유실)
   - **값의 범위가 넘치면 엉뚱한 값이 나올 수 있음**

### Scanner 클래스
- **패키지**: `java.util.Scanner`
- **역할**: 키보드 등으로부터 데이터를 입력받기 위해 사용함

**주요 메서드**
1. `next()`: 공백 기준으로 문자열 입력 (단어 단위)
2. `nextLine()`: 줄바꿈 기준으로 문자열 입력 (문장 단위)
3. `nextInt()`, `nextDouble()`: 해당 타입에 맞는 숫자 입력

**주의사항**
- 숫자 입력(`nextInt` 등) 후 문자열 입력(`nextLine`) 시 버퍼에 남은 엔터 처리 필요
- 사용이 끝나면 `sc.close()`를 호출하여 리소스를 반환하는 것이 권장됨
