## 클래스
클래스는 데이터와 그 데이터를 처리하는 메서드를 하나의 단위로 묶어 관리하는 사용자 정의 데이터 형식
객체 지향 프로그래밍에서 특정 상태를 저장하는 변수와 동작을 수행하는 메서드를 설계하는 역할

### 클래스 정의 및 인스턴스 생성
클래스는 class 키워드를 사용하여 선언
클래스를 호출하여 메모리에 실제 객체를 생성하는 과정을 인스턴스화라고 하며 생성된 객체를 인스턴스라고 함

### 생성자와 인스턴스 변수
생성자인 init 메서드는 인스턴스가 생성될 때 속성값을 초기화하기 위해 자동으로 호출
인스턴스 변수는 self 키워드를 통해 정의하며 인스턴스마다 독립적인 데이터를 유지

### 인스턴스 메서드
인스턴스 메서드는 특정 인스턴스의 상태를 변경하거나 정보를 확인하는 함수
메서드 정의 시 첫 번째 매개변수로 현재 인스턴스 자신을 가르키는 self를 반드시 명시해야함

### 클래스 변수
클래스 변수는 클래스 블록 내부에서 직접 선언하며 모든 인스턴스가 단일한 메모리 주소를 참조하여 값을 공유
특정 개별 인스턴스가 아닌 클래스 전체에 속한 데이터를 관리할 때 사용

### 매직 메서드
매직 메서드는 클래스 내부에 정의되어 특정 상황에서 파이썬 인터프리터에 의해 자동으로 호출되는 특수 메서드
메서드 이름 앞뒤에 이중 언더바가 붙는 형식을 가지며 사용자 정의 객체에 파이썬 내장 기능을 구현할 때 많이 사용
(객체의 문자열 표현 및 정리)
str 메서드는 객체를 문자열로 변환할 때 호출되며 len 메서드는 len 함수를 호출할 때 객체의 길이를 반환

### 비교 연산자 정의
eq, lt 등의 메서드를 정의하여 객체 간의 비교 연산 동작을 직접 설정할 수 있음

### 소멸자 메서드
del 메서드는 객체의 참조 횟수가 0이 되어 메모리에서 해제되기 직전에 호출되는 특수 메서드
파일 닫기나 네트워크 접속 종료와 같은 정리 작업을 수행할 때 사용


## 데코레이터
데코레이터는 함수나 메서드의 소스 코드를 직접 수정하지 않고기능을 추가하거나 변경하기 위해 사용하는 고차 함수
기호를 사용하여 정의된 기능을 대상 코드의 상단에 배치하는 구문을 가짐

### 데코레이터의 기본 구조
데코레이터는 함수를 매개변수로 전달받아 내부에 새로운 함수를 정의한 뒤 이를 반환하는 구조를 취함
클래스 내부의 메서드에 적용할 경우 첫 번째 매개변수로 인스턴스 자신인 self를 전달받는 구조를 반영해야함


## 고차 함수
고차 함수는 함수를 인자로 받거나 함수를 반환하는 함수를 정의
일급 객체의 특성을 이용하여 논리적인 연산 단위를 유연하게 결합하는 메커니즘

### 함수를 매개변수로 전달받는 고차 함수
특정 동작을 수행하는 함수를 인자로 받아 내부에서 실행함으로써 로직의 중복을 제거하고 확정성을 확보


## 일급 객체
일급 객체는 프로그래밍 언어에서 다음 세 가지 조건을 충족하는 개체를 의미
변수나 데이터 구조에 할당할 수 있어야 함
함수의 배개변수로 전달할 수 있어야 함
함수의 반환값으로 사용할 수 있어야 함
파이썬에서 함수는 일급 객체로 취급되므로 정수나 문자열과 동일하게 취급

### 변수 할당 및 데이터 구조 저장
함수를 변수에 할당하여 호출하거나 리스트, 딕셔너리 등의 데이터 구조에 저장하여 관리


## 클래스의 속성 관리
데코레이터는 함수나 메서드의 동작을 수정하거나 확정하기 위해 사용하는 특수 구문
클래스 내부에서 속성 접근 제어와 메서드의 성격을 규정하는데 사용

### property 접근자
property 데코레이터는 메서드를 호출할 때 괄호를 생략하고 변수처럼 접근할 수 있게 함
내부 데이터를 직접 노출하지 않고 가공된 값을 반환하거나 읽기 전용 속성을 만들 때 사용

### setter 설정자
setter 데코레이터는 인스턴스 변수에 값을 할당할 때 실행되는 로직을 정의
데이터 할당 전 유효성 검사를 수행하여 부적절한 데이터가 저장되는 것을 방지

### 클래스 메서드
classmathod 데코레이터는 인스턴스가 아닌 클래스 자체를 첫 번째 매개변수로 전달받음
클래스 변수에 접근하거나 클래스 상태를 변경하는 로직을 구현할 때 사용

### 정적 메서드
staticmethod 데코레이터는 인스턴스나 클래스에 대한 참조 없이 동작하는 메서드를 정의
클래스의 네임스페이스 안에 포함되어 있으나 클래스 내부 상태와 무관한 독립적인 기능을 수행
